# Tactical Auto-Battler (React Application)

Веб-застосунок для розстановки військ і покрокової симуляції бою між армією гравця та ворожою армією. Користувач обирає юнітів, розміщує їх на полі, запускає бій і спостерігає лог подій у реальному часі.

Розроблено в рамках Лабораторної роботи 6 (React Application).

## Основний функціонал

- Інтерактивна фаза підготовки: вибір юніта з панелі та розміщення на полі 3x5.
- Автогенерація ворога: формування ворожої армії під кількість юнітів гравця.
- Покроковий бій: сторони атакують по черзі, по одній дії за тік.
- Коректний таргетинг: атака виконується по найближчій живій цілі в активному ряду.
- Журнал бою: вивід усіх атак із розрахованим уроном і множниками типів.
- Керування швидкістю: слайдер темпу бою та пресети Повільно, Нормально, Швидко.
- Екран результату: фіксація перемоги або поразки з можливістю перезапуску.

## Стек технологій

- Frontend: React (функціональні компоненти, Hooks)
- Збірка: Vite (швидкий HMR та оптимізований білд)
- Стилізація: CSS utility класи (Tailwind-підхід)
- Архітектура: поділ на UI-компоненти та інкапсульовану бізнес-логіку в кастомному хуку

## Структура проєкту

Проєкт дотримується підходу Separation of Concerns: відображення винесено в компоненти, логіка бою інкапсульована в hooks.

```text
src/
├── components/
│   ├── Battlefield.jsx    # Відображення поля бою (ряди гравця та ворога)
│   ├── BattleLog.jsx      # Журнал бойових подій
│   ├── SetupPanel.jsx     # Панель вибору юнітів
│   └── UnitCard.jsx       # Картка юніта/слоту
├── hooks/
│   └── useBattle.js       # Кастомний хук: стан бою, тики, черга ходу, фінальний статус
├── models/
│   └── units.js           # Типи юнітів, параметри та матриця множників шкоди
├── utils/
│   └── aiLogic.js         # Генерація ворожої армії
├── App.jsx                # Головний компонент-оркестратор фаз гри
├── main.jsx               # Точка входу застосунку
├── App.css                # Стилі компонента App
└── index.css              # Глобальні стилі
```

## Як користуватися застосунком

Після запуску локального сервера виконайте такі кроки:

1. Фаза розстановки:
- Оберіть тип юніта в Setup Panel.
- Клікніть по клітинці на полі Твої війська, щоб поставити юніта.
- За потреби натисніть Очистити поле.

2. Запуск бою:
- Натисніть Почати бій.
- Ворожа армія згенерується автоматично.

3. Спостереження за боєм:
- Поле оновлюється покроково.
- У Battle Log відображаються всі атаки та розрахований урон.

4. Керування швидкістю:
- Використовуйте слайдер Швидкість бою для тонкого налаштування інтервалу тіку.
- Або натискайте пресети Повільно, Нормально, Швидко для швидкого перемикання.

5. Завершення:
- Після перемоги або поразки відображається екран результату.
- Натисніть Спробувати знову, щоб розпочати новий матч.

## Встановлення та запуск локально

Склонуйте репозиторій:

```bash
git clone https://github.com/Kryvutsia-Roman/Tactical-Auto-Battler.git
```

Перейдіть у директорію проєкту:

```bash
cd lab6
```

Встановіть залежності:

```bash
npm install
```

Запустіть dev-сервер:

```bash
npm run dev
```

Відкрийте адресу з терміналу (зазвичай):

```text
http://localhost:5173
```

## Production build

Для збірки production-версії виконайте:

```bash
npm run build
```

Для локального перегляду production-збірки:

```bash
npm run preview
```
